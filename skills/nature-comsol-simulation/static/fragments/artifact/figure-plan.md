# Artifact: Figure Plan

Plan figures as evidence chains, not as decorative COMSOL screenshots.

Prioritize panels that connect model setup to claim:

1. Geometry, boundary conditions, loads, and material regions.
2. Mesh and local refinement around claim-critical regions.
3. Primary field result with scale bar, units, colorbar, and comparable range.
4. Quantitative extraction from regions/paths, not only contour plots.
5. Mesh or time-step convergence for claim-relevant outputs.
6. Sensitivity/parameter sweep or validation panel.
7. Claim summary with uncertainty or boundary statement.

Risk checks:

- Same color scale for comparative contour panels.
- Avoid cropped contours that hide boundary conditions or load path.
- Do not use a local peak as the only evidence for a broad design claim.
- Include failed/unstable cases if they define the model's boundary.
