# Artifact: Response Strategy

Use for reviewer comments about model validity, mesh convergence, boundary
conditions, validation, contact/coupling, or overclaiming.

Structure:

1. Acknowledge the exact concern without over-conceding.
2. Separate evidence already present from new analyses to be added.
3. Describe concrete additions: mesh/time-step convergence, sensitivity,
   benchmark, validation, figure revision, Methods expansion, or wording change.
4. Draft a reply paragraph using only supplied facts.
5. List manuscript edits and remaining risks.

Do not invent completed analyses. If the user has not performed the extra test,
write it as "we will add" or "we propose to add", not "we have shown".
