# Artifact: Methods

Write Methods so a reviewer can reconstruct the model contract.

Include:

- COMSOL version, modules, physics interfaces, study type, and solver.
- Geometry source, dimensionality, symmetry, simplifications, and coordinate/unit
  conventions.
- Material laws, parameters, units, and parameter sources.
- Boundary conditions, loads, constraints, initial conditions, contact/friction,
  thermal inputs, and coupling interfaces.
- Mesh type/order, refinement strategy, mesh-convergence method, and final mesh.
- Solver tolerances, nonlinear/time-step/coupling settings, and convergence
  criteria.
- Outputs, post-processing, extracted regions, parameter sweeps, and validation.

If details are missing, draft a Methods skeleton with bracketed placeholders and
list the missing inputs rather than guessing.
